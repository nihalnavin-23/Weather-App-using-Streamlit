🌤️ Weather App
A simple and elegant weather application built with Streamlit that uses the Weather Stack API to fetch and display real-time weather information for any city.

✨ Features
Real-time Weather Data: Fetches current weather conditions including temperature, humidity, wind speed, and more.

User-Friendly Interface: Clean and intuitive design with clear metrics and visual indicators.

Search by City: Enter any city name worldwide to get its weather.

Weather Icons: Displays weather condition icons for quick visual reference.

Detailed Information: Shows temperature, feels-like temperature, humidity, wind speed, cloud cover, and visibility.

Location Details: Displays city, country, region, and local time.

Raw JSON Viewer: Expandable section to view the complete API response.

Responsive Design: Works well on both desktop and mobile devices.
